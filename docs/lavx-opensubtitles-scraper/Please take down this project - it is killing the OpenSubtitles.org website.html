    Please take down this project — it is killing the OpenSubtitles.org website#9
    Copy link

    Description

    opensubtitles
    opened on Mar 19, 2026
    Issue body actions

    Hello,

    I am writing to ask you to take down this project, or at the very least, stop distributing it publicly.

    What this scraper does to our server:

    Your tool makes automated HTTP requests directly to www.opensubtitles.org at machine speed, bypassing our Cloudflare protection using cloudscraper. It is designed explicitly to avoid our rate limiting and anti-bot measures.

    Every user who runs your Docker image generates a continuous stream of requests against our infrastructure — at zero cost to them, and at direct cost to us in server load, bandwidth, and Cloudflare billing. With 31 stars and growing usage, this is not a single actor problem — it scales with every new user who runs your container.

    Why this is harmful:

    OpenSubtitles.org is a free, community-supported service. We provide a legitimate, documented REST API at api.opensubtitles.com with generous free tier limits specifically so that tools like Bazarr can integrate properly — without scraping. Your project bypasses this entirely, giving your users free access while undermining the sustainability of the service for everyone else.

    Scraping our HTML pages also breaks every time we update our frontend, causing your users to file bugs and adding to your own maintenance burden.

    What we ask:

    Take down the Docker image
    Archive or remove this repository
    If you genuinely want to help Bazarr users — contribute to the official OpenSubtitles Bazarr provider which uses our proper API
    We have a free API tier. There is no justification for scraping.

    Thank you.
